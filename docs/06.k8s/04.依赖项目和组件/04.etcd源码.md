* 简单命令的使用 https://zhuanlan.zhihu.com/p/104208137
``` bash
export ENDPOINTS="192.168.56.110:2379"
cd /Users/hfb/projects/go/mod-pro/etcd/bin
./etcdctl --write-out=table --endpoints=$ENDPOINTS endpoint status
```
