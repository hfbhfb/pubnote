## NVM – Install and Manage Multiple Node.js Versions in Linux



## 步骤1
  * wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
## 步骤2 查看上一步是否成功
  * bash 重连 `重点强调,需要重连`
  * nvm  
  * nvm ls
  * nvm ls-remote
## 步骤3 安装和使用
  * nvm install v14.19.1
  * nvm use v14.19.1
  * nvm install v16.14.2
  * nvm use v16.14.2
