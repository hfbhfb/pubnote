## go语言历史下载版本
  * https://go.dev/doc/install
  * 历史版本
    * https://go.dev/dl/
  * wget https://dl.google.com/go/go1.16.15.linux-amd64.tar.gz


## 安装对应版本
  * rm -rf /usr/local/go && tar -C /usr/local -xzf go1.16.15.linux-amd64.tar.gz
  * export PATH=$PATH:/usr/local/go/bin

## 验证
  * go version
