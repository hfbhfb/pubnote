* redis 基本命令
  * type 2
    * 获取key对应的类型
  * hgetall 2  
    * hash 类型获取所有
  * info
    * 获取redis信息
    * used_memory_human:374.01M
    * db0:keys=4579461,expires=0,avg_ttl=0
    * maxmemory_human:1.95G
  * 清空所有redis (FLUSHALL)
    * FLUSHALL
  * (redis 日志分析)
    * monitor
